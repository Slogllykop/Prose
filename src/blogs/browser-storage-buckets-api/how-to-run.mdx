### How to Run This Locally

1. **Prerequisites**: Ensure you are using **Chrome 122+** or **Edge 122+**.
2. **Setup**: Clone the [Storage-Bucket-API-Example](https://github.com/Slogllykop/Storage-Bucket-API-Example) repository or replicate the folder structure above.
3. **Local Server**: The Storage Buckets API requires a **Secure Context**. Run a local server from the root directory:

   **Using Node.js:**
   ```bash
   npx serve .
   ```

   **Using Python:**
   ```bash
   python -m http.server 8000
   ```

4. **Inspect**: Open DevTools, go to the **Application** tab, and look for the **Storage Buckets** tree item to see the API in action.
